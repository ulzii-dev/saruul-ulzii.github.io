Question 1 What is the difference between the following 2 statements? 
        setTimeout(booyah, 2000);
        setTimeout(booyah(), 2000);
      
Answer: 
        1. setTimeout(booyah, 2000)  is calling setTimeout properly and booyah function will run after 2000ms.
        2. setTimeout(booyah(), 2000) is calling booyah function directly, and it won't work after 2000ms.
