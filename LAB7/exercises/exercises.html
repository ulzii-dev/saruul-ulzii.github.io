Question 9 Write Javascript code for creating three Employee objects using the "new" keyword and a constructor function. Employee objects have the following fields: name, salary, position.

Question 10 Write a Javascript function that takes any number of input arguments and returns the product of the arguments.

Answer:

        const prodIt = function productFinder() {
            var product = 1;
            for (var i = 0; i < arguments.length; i++) {
                product *= arguments[i];
            }
            return product;
        }
