Question 1: In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”? 

      
Answer:

      
        JQuery function
        JQuery function is pre-defined, usable functions. Easy to use and it will save time and line count.
        

        JQuery object
        JQuery object is object that returned from JQuery function. Easy to access its dom elements, nodes.
