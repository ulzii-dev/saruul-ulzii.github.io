Answer:

        Both will give same result, because $.each is structured like that: function(index, element)
        Both parameters are optional, 1st one is index, 2nd one is element, it's same like this inside this function.

        // code fragment 1
        $("li").each(function(idx, e) { <= All params passed, index and element
            $(e).css(“color”, “yellow”); 
        });

        // code fragment 2
        $("li").each(function() {       <= No param passed, index and element are optional, in this case, it used this, to access certain JQuery object, in this case LI element
            $(this).css(“color”, “yellow”); 
        });

        // code fragment 3
        $("li").each(function(idx) {    <= 1st param passed, its index, and didn't use in this function
            $(this).css(“color”, “yellow”); 
        });
